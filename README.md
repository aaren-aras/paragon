# PARAGON

## GDExtension
This project uses **GDExtension** to compile C++ code into a dynamic library that Godot loads at runtime,
letting me work on the game in C++ without the trouble of recompiling the whole engine every time I make
changes. This'll be one of several projects I use to practice C++ (*even if it ends up being overkill*). 

To build the project:
```scons platform=windows target=template-debug```
